GA Aware

GA Aware provides better Google Analytics reporting and tracks more user activity without writing code - Event tracking, outbound, asset, mailto links and more.

Download this project as a .zip file Download this project as a tar.gz file

What is it

Through years of building websites, installing Google Analytics manually tracking events, assets, setting up cross domain analytics, etc, Clockwork Active Media Systems decided to write a plugin to make it easier, and better yet automatically configure itself using common requests we get and recommendations we give.

View Documentation

Default Installation

Just set the Google Analytics UA code and you're good to go!

  • Links to images, documents, videos, music, spreadsheets, zip, flash and many other asset files
  • Outbound links
  • Jump links (starting with a #)
  • Mailto links
  • Links to FTP, file, phone number and Spotify
  • Forms that submit to an external domain
  • Send data to multiple UA codes at once
  • Ignore development clones of your site without changing code
  • Recognize links added through AJAX
  • Ignore links by adding a css class
  • Use link title for event name rather than URL

Additional features

These features can generally be added and configured with one line of code!

  • Cross domain tracking
  • Social events from facebook and twitter widgets
  • Configurable event taxonomy
  • Set custom variables
  • E-Commerce tracking
  • Switch between tracking as virtual pageview or events
  • Methods for tracking to all active code with one call
  • Debug mode

Browser support

  • Internet Explore 6+
  • Chrome
  • Firefox
  • Opera
  • Safari

New Features in v1.9

v1.9 is currently in beta testing

  • Seamless integration with multiple Javascript libraries installed
  • Track right clicks and maintains shift or control clicking links
  • Pageviews don't count as bounces after 30 sec.
  • Session won't timeout if a page is active for 30 min.
  • Default custom variables set for "Day of Week" & "Hour of Day"
